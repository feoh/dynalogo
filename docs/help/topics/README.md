# Help topics

Each Markdown file below this directory is a canonical help topic.

Start new topics by copying [`_template.md`](_template.md), then place the file
under the most specific category directory:

- `primitives/`
- `library-procedures/`
- `data/`
- `syntax/`
- `concepts/`
- `frontends/`
- `compatibility/`

See [`../topic-schema.md`](../topic-schema.md) for required fields and
validation rules.
